Overview

Purpose

This website will be a competetor to cstimer.net so it will be giving Rubiks cube scrambles and recording the solve times. I want to support atleast a few types namely the 3x3, 4x4, and 5x5. Also because this is being developed with mobile screen in mind this could be a sold option for using your phone as a timer

Audience

This will be used by cubers who like recording a history of their solve times.

Dynamic elements

JS will update the current scramble to be used and display it for the user. JS will also update a timer in the middle of the screen to show how long a solve took. There will also be a box with the solve history and JS will allow toggling between solve date or fastest solve as the ordering method. The JS will also be listening for the spacebar to be pressed or the screen to be tapped to start the timer. JS will also be generating the graph on the other page.
